# Web HMI 网页技术栈

本章节介绍面向 SynapticSYS 的 Web HMI 技术栈选型与工程实践建议，涵盖 Web SCADA 技术调研、主流方案对比，并结合基于ROS2开发的项目目标给出技术方向与指导。

## Web HMI 技术调研

### 1. 行业概述

Web HMI（Human-Machine Interface）是工业自动化领域中通过网页技术实现的人机交互界面，用于监控和控制工业设备与系统。它继承了传统 HMI 的功能，同时具备 Web 技术的灵活性和跨平台特性。

主要应用场景：
- 工业自动化控制系统
- 设备远程监控与管理
- 生产数据可视化与分析
- 能源管理系统（EMS）
- 智能工厂/智能制造

### 2. 主流技术方案

#### 西门子 Web HMI 方案
- **技术栈**：Siemens WinCC Web Navigator、Siemens Industrial Edge
- **核心特性**：
  - 与西门子 PLC 深度集成
  - 实时数据采集与可视化
  - 完善的安全机制
  - 支持边缘计算
- **适用场景**：大型工业自动化项目，对可靠性要求高

#### Web SCADA 技术
- **代表产品**：Ignition SCADA、Inductive Automation
- **技术栈**：Java、HTML5、WebSocket
- **核心特性**：
  - 跨平台部署
  - 强大的数据处理能力
  - 丰富的可视化组件
  - 支持多种工业协议
- **适用场景**：中型工业自动化项目，需要灵活扩展

#### 轻量级 Web HMI 方案
- **代表技术**：Node-RED + Dashboard、MQTT.js + Vue/React
- **技术栈**：JavaScript/TypeScript、WebSocket、MQTT
- **核心特性**：
  - 快速开发
  - 低成本部署
  - 社区活跃
- **适用场景**：小型项目或原型验证

## 技术栈对比

| 技术方案 | 技术栈 | 性能 | 可扩展性 | 开发效率 | 集成能力 | 适用场景 |
|---------|--------|------|----------|----------|----------|----------|
| 西门子方案 | 专有技术 | 高 | 中 | 中 | 高（西门子生态） | 大型工业项目 |

## 结合项目目标的技术方向

### 1. 核心技术选型建议

- **前端框架**：React + TypeScript
  - 生态成熟，社区活跃
  - 组件化开发，易于维护
  - 性能优秀，适合复杂可视化

- **实时通信**：WebSocket + ROS2 网关
  - WebSocket 提供浏览器与服务器的实时连接
  - ROS2 网关实现与后端ROS2系统的集成，支持话题、服务、动作等ROS2通信机制
  - 支持断线重连与数据缓存

- **可视化库**：
  - ECharts：图表可视化
  - SVG/Canvas：自定义图形与动画
  - Three.js：3D设备模型可视化

- **后端技术**：.NET Core + SignalR
  - 与项目核心技术（C#）保持一致
  - SignalR 提供强大的实时通信能力
  - 支持微服务架构

### 3. 关键功能实现

- **实时数据监控**：
  - 使用 WebSocket 实现数据推送
  - 通过 ROS2 话题订阅实时数据
  - 支持数据更新频率配置
  - 异常数据报警与处理

- **可视化展示**：
  - 设备状态仪表盘
  - 实时趋势图表
  - 3D设备模型与动画
  - 流程可视化
  - ROS2 话题数据可视化

- **远程控制**：
  - 设备操作权限管理
  - 通过 ROS2 服务/动作实现远程控制
  - 操作日志记录
  - 紧急停止机制

- **历史数据查询**：
  - 时间范围查询
  - 数据导出（CSV、PDF）
  - 数据分析与报表
  - 与 ROS2 数据存储集成

## 开发与部署建议

### 1. 开发流程

- 使用模块化开发，组件复用
- 采用CI/CD流程，自动化测试与部署
- 建立完善的文档与API规范

### 2. 部署架构

- 容器化部署（Docker + Kubernetes）
- 边缘部署支持（轻量级容器）
- 负载均衡与高可用性设计

### 3. 性能优化

- 前端性能优化（组件懒加载、代码分割）
- 数据压缩与缓存
- 合理的更新频率设置
- 服务器性能监控与调优

## 安全策略

- 身份认证与授权（OAuth2/OIDC）
- 传输加密（TLS）
- 设备访问控制
- 操作审计日志
- 防跨站攻击（XSS、CSRF）
