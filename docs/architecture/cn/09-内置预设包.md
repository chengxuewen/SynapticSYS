# 内置预设包设计

## 1. 设计目标

- 提供基于ROS2的内置预设包，方便开发者快速构建应用
- 支持多种编程语言（Python、C++），遵循ROS2的多语言支持
- 与ROS2生态系统无缝集成，利用ROS2的标准API
- 遵循ROS2的模块化设计原则，便于扩展和维护
- 保持与ROS2 API的一致性，降低学习曲线

## 2. 目录结构

内置预设包采用ROS2标准的Colcon/Ament包结构，存放在项目根目录的`src/`目录中：

```
SynapticSYS/
├── src/                      # ROS2包目录
│   ├── synaptic_core/        # 核心功能包
│   │   ├── package.xml
│   │   ├── CMakeLists.txt
│   │   ├── include/synaptic_core/
│   │   └── src/
│   ├── synaptic_ros2/        # ROS2集成包
│   │   ├── package.xml
│   │   ├── setup.py
│   │   └── synaptic_ros2/
│   ├── synaptic_msgs/        # 消息定义包
│   │   ├── package.xml
│   │   ├── CMakeLists.txt
│   │   └── msg/
│   └── synaptic_utils/       # 工具函数包
│       ├── package.xml
│       ├── setup.py
│       └── synaptic_utils/
└── colcon.pkg                # Colcon配置文件
```

## 3. 内置包列表

### 3.1 synaptic_core

核心功能包，提供基础功能：
- 设备管理
- 系统监控
- 配置管理
- 核心服务接口

### 3.2 synaptic_ros2

ROS2集成包，提供Python接口：
- `synaptic_ros2.Node` - 扩展的ROS2节点类
- `synaptic_ros2.Publisher` - 增强的发布者
- `synaptic_ros2.Subscriber` - 增强的订阅者
- `synaptic_ros2.Service` - 服务实现
- `synaptic_ros2.ServiceClient` - 服务客户端

### 3.3 synaptic_msgs

消息定义包，包含扩展的ROS2消息类型：
- 设备状态消息
- 系统监控消息
- 配置管理消息
- 与标准ROS2消息的兼容层

### 3.4 synaptic_utils

通用工具包，提供各种辅助功能：
- 时间工具（与ROS2 Time兼容）
- 日志工具（与ROS2 Logger集成）
- 配置管理（支持YAML/JSON）
- 类型转换（ROS2消息与原生类型）
- 字符串处理

## 4. 实现机制

### 4.1 包发现

内置包通过ROS2的Colcon构建系统自动发现：
- 遵循ROS2的包结构规范
- 利用Colcon的包发现机制
- 支持从`src/`目录自动识别所有ROS2包

### 4.2 构建流程

内置包使用ROS2标准的Colcon构建流程：
- `colcon build` 命令构建所有包
- 自动处理包间依赖关系
- 支持并行构建以提高效率
- 构建产物安装到`install/`目录

### 4.3 环境变量

内置包构建完成后，通过ROS2的环境设置脚本自动配置：
- `source install/setup.bash`（Linux/macOS）
- `install/setup.ps1`（Windows）
- 自动设置`PYTHONPATH`、`LD_LIBRARY_PATH`等环境变量

## 5. ROS2集成

内置预设包与ROS2深度集成：
- 直接使用ROS2的核心API和数据结构
- 支持ROS2的标准消息类型
- 与ROS2的工具链（colcon、rosdep等）兼容
- 可与其他ROS2包无缝交互

## 6. 扩展机制

内置预设包支持ROS2标准的扩展方式：
- 通过创建新的ROS2包扩展功能
- 支持ROS2的插件系统
- 允许通过launch文件配置和启动
- 兼容ROS2的参数系统

## 7. 测试策略

内置预设包使用ROS2标准的测试框架：
- 单元测试：使用`gtest`（C++）和`pytest`（Python）
- 集成测试：使用ROS2的`launch_testing`
- 功能测试：使用ROS2的`rostest`
- 性能测试：使用ROS2的性能测试工具
